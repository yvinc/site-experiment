Welcome to my experimental site!

To create this page, add a markdown .md file into content/ folder.

You can adjust the cover image by changing the link in the cover link in markdown’s format section.

You can adjust the type of the markdown file by changing the type in the format section.

Currently, this section is set to type: 'slide'. You can swipe to left to display the second slide!

Posts in .md are placed within content/posts/. You can adjust their categories = format entry to display them in the Categories slider right below this slide.

Please see themes/hugo-brewm/exampleSite/ folder structure for reference.
